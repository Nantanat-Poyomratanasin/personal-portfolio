# Portfolio Website Documentation

## Overview

This project is a personal portfolio website with a pixel-art inspired theme.  
The website contains 2 main pages:

1. Home Page
2. About Page

Purpose:

- Showcase profile and resume
- Display projects
- Show skills and work experience
- Provide contact and social links

Theme:

- Pixel Art
- Minimal layout
- Soft grayscale design with retro-inspired style

---

# Website Structure

## Navigation Bar

Appears on every page.

### Sections

- Logo / Name
- About
- Contact

---

# Home Page

## Purpose

The landing page introduces the owner and highlights featured projects.

---

## Home Page Layout

### 1. Hero Section

Contains:

- Short introduction / summary
- Resume link
- Profile picture

Example:

- Name
- Short bio
- Resume download button

---

### 2. Featured Projects Section

Displays project cards.

Each project card includes:

- Project image
- Project title
- Short description
- Project link

Example projects:

- Project 1
- Project 2

---

### 3. Social Links Section

Contains:

- LinkedIn
- GitHub

Purpose:

- Allow recruiters or visitors to connect

---

# About Page

## Purpose

Show detailed personal information and background.

---

## About Page Layout

### 1. Skills Section

Displays multiple skill cards/icons.

Example:

- Backend
- UI/UX

Possible technologies:

- HTML
- CSS

- React

- PostgreSQL

---

### 2. Work Experience Section

Contains:

- Company/role information
- Work descriptions

Possible structure:

- Job title

---

### 3. Certificate Section

Displays certificates in slider/carousel format.

Features:

- Left/right navigation arrows
